# ぐるめフレンズ

すごーい！あなたはグルメなフレンズなんだね！

## システム要求
- ruby: 2.4.0
- rails: 5.0.2

# ビルド

```
ビルド
$ docker-compose build
起動
$ docker-compose up -d
停止
$ docker-compose down -v
状態確認
$ docker-compose ps
```
